# rsweb-tmpl
Template for full-stack Web development in Rust using Warp and Maud with HTTPS support.

### Features
- [x] HTTPS support
- [x] Database support
- [x] Cache support
- [x] Server-side rendering
- [x] Client-side routing
- [x] Token authentication
- [x] Google OAuth2
- [x] REST API
- [x] Database migrations
- [x] Database seeding
- [x] Theming
- [x] User login
- [ ] User registration (Partial)
- [x] Static assets
- [x] Blog
